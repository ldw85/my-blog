---

## 一、项目技术架构概览

### 1.1 技术栈
- **框架**：Astro 5.0 + Tailwind CSS 3.4
- **语言**：TypeScript
- **部署支持**：Vercel、Netlify、Cloudflare Pages、Docker

### 1.2 目录结构
```
my-blog/
├── src/
│   ├── components/      # 组件库
│   │   ├── blog/        # 博客相关组件
│   │   ├── common/      # 通用组件
│   │   ├── ui/          # UI 组件
│   │   └── widgets/     # 页面小部件
│   ├── layouts/         # 布局组件
│   ├── pages/           # 页面路由
│   │   ├── [...blog]/   # 博客系统
│   │   ├── homes/       # 多种首页模板
│   │   └── landing/     # 落地页模板
│   ├── data/post/       # 文章内容 (Markdown)
│   ├── config.yaml      # 站点配置
│   └── navigation.ts    # 导航配置
├── public/              # 静态资源
└── wrangler.toml        # Cloudflare Pages 配置
```

## 九、总结
